Setup guide
1
Install Cloudflared
Download cloudflared on the machine running Revit. Quick install on Windows:
winget install Cloudflare.cloudflared
2
Start Revit & load the plugin
Open Revit, load the DeepBIM plugin. Confirm it is listening on port 9080.
3
Start the tunnel
Run in a terminal on the Revit machine:
cloudflared tunnel --url http://localhost:9080
4
Copy the generated URL
After a few seconds, cloudflared prints a public URL:
https://xxxx.trycloudflare.com
Copy it and paste into the form on the right →
Step 5 — Register URL
Active URL
{{CURRENT_URL_DISPLAY}}
Every time you restart the Cloudflare tunnel, a new URL is generated. Simply paste the new URL here and click Connect — no server restart needed.
